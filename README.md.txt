# Responsive HTML Email Template

A table-based responsive HTML email template built using HTML5 and inline CSS.

## Features
- 600px centered email layout
- Table-based structure (email-client safe)
- Inline CSS styling
- Cross-client compatibility approach
- Two-column feature section
- CTA button

## Email Best Practices Used
- Fixed width container (600px)
- Inline CSS (for better client support)
- Table-based layout instead of div/flexbox
- Role="presentation" for accessibility
- Cross-browser rendering consideration

Open index.html in a browser or send to your Gmail for testing.
