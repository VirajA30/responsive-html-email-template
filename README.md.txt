# Responsive HTML Email Template

A responsive, table-based HTML email template built with HTML5 and inline CSS, designed for compatibility across major email clients (Gmail, Outlook, mobile).

## Features

- 600px centered email layout
- Inline CSS for maximum client support
- Table-based structure (email-safe)
- Two-column feature section
- Call-to-Action button
- Compatible with desktop & mobile devices

## How to View

1. Clone or download the repository
2. Open `index.html` in a browser
3. Or send the file to your email to preview Gmail/Outlook rendering

## Best Practices Used

- Table-based layout instead of div/flexbox
- Inline CSS for reliable email client rendering
- Role presentation set for accessibility
- Email-safe font and spacing
